How Hangul Works

The Key Insight

Korean is written in syllable blocks. Each block contains 2-4 characters arranged in a specific pattern based on the vowel's shape.

Did you know? The world's only designed alphabet

Unlike almost every other alphabet that evolved slowly over centuries, Hangul was invented in 1443 by King Sejong the Great and a team of scholars. It was designed from scratch to be easy to learn.

King Sejong's goal was literacy for common people. Legend says he wanted something so simple that "a wise man could learn it in a morning, and even a fool could learn it in ten days."

The original name was Hunminjeongeum — "Proper Sounds to Instruct the People."

Sources: Britannica, Wikipedia

Vowel Orientation Rule

The vowel's shape determines how the block is arranged:

Vertical Vowels

ㅏ ㅓ ㅣ ㅐ ㅔ

Tall vowels go to the right of the consonant

가
ㄱ + ㅏ
너
ㄴ + ㅓ
미
ㅁ + ㅣ

Horizontal Vowels

ㅗ ㅜ ㅡ

Wide vowels go below the consonant

고
ㄱ + ㅗ
누
ㄴ + ㅜ
그
ㄱ + ㅡ

Block Patterns

2-character blocks

가 = consonant + vertical vowel
고 = consonant + horizontal vowel

3-character blocks

간 = consonant + vowel + final consonant
곤 = consonant + vowel + final consonant

The Silent Placeholder: ㅇ

Every syllable block needs a consonant. When a syllable starts with a vowel sound, ㅇ is used as a silent placeholder.

아
= "a" (ㅇ is silent)
오
= "o" (ㅇ is silent)
이
= "i" (ㅇ is silent)

Note: ㅇ only has a sound ("ng") when it appears at the end of a syllable.
