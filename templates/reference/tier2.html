Extensions

Building on the basics with Y-vowels and special rules.

The ㅇ Rules

Initial Position: Silent

아 어 오

When ㅇ starts a syllable, it makes no sound. It's just a placeholder because every syllable needs a consonant.

Final Position: "ng"

강 성 영

When ㅇ ends a syllable (as batchim), it sounds like 'ng' in "sing".

Y-Vowels (4)

Pattern: Add an extra line to the basic vowel = add a "y" sound at the start.

Letter Sound Base Hint
ㅑ ya ㅏ + line Like 'ya' in yacht
ㅕ yeo ㅓ + line Like 'yu' in yuck
ㅛ yo ㅗ + line Like 'yo' in yoga
ㅠ yu ㅜ + line Like 'you'

The "Eh" Vowels

Good news: These two sound almost identical in modern Korean. Don't stress about the difference!

Letter Sound Hint
ㅐ ae Like 'a' in can or 'e' in bed
ㅔ e Like 'e' in bed (same as ㅐ in practice)
